# TUX-GAME-JAVA-XML

Projet universitaire réalisé en **Licence 3 Informatique MIAGE**.  
Il s’agit d’un **jeu de type Scrabble en 3D**, développé en **Java**, utilisant la bibliothèque **Env3D** pour le rendu graphique.

Le projet met volontairement l’accent sur l’utilisation avancée des **technologies XML**, incluant :
- la validation des documents XML à l’aide de **schémas XSD** ;
- la transformation des données XML via des **feuilles XSLT** ;
- le parsing XML à l’aide des **parsers SAX et DOM**.
